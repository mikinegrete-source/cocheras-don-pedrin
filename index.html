// ═══════════════════════════════════════════════════════════════ // COCHERAS DON PEDRÍN — Google Apps Script v6 // Borrá el código anterior y pegá todo este // ═══════════════════════════════════════════════════════════════ const SHEET_NAME = 'Cocheras'; function getSheet() { const ss = SpreadsheetApp.getActiveSpreadsheet(); let sheet = ss.getSheetByName(SHEET_NAME); if (!sheet) { sheet = ss.insertSheet(SHEET_NAME); sheet.getRange(1,1,1,11).setValues([['N°','NOMBRE','PATENTE','VEHÍCULO','TELÉFONO','COCHERA','HORARIO','LLAVE','ESTADO','MONTO','OBSERVACIONES']]); sheet.getRange(1,1,1,11).setFontWeight('bold'); sheet.setFrozenRows(1); } return sheet; } function responder(data) { return ContentService .createTextOutput(JSON.stringify(data)) .setMimeType(ContentService.MimeType.JSON); } function doGet(e) { return responder({ ok: true, mensaje: 'API Cocheras Don Pedrín v6 activa' }); } function doPost(e) { try { let body; // Acepta tanto JSON directo como form POST (campo postData) if (e.postData && e.postData.type === 'application/json') { body = JSON.parse(e.postData.contents); } else if (e.parameter && e.parameter.postData) { body = JSON.parse(e.parameter.postData); } else if (e.postData && e.postData.contents) { body = JSON.parse(e.postData.contents); } else { return responder({ ok: false, error: 'Sin datos' }); } if (body.action === 'guardarTodo') { const clientes = body.clientes; if (!clientes || !Array.isArray(clientes)) { return responder({ ok: false, error: 'Sin clientes' }); } const sheet = getSheet(); const lastRow = sheet.getLastRow(); if (lastRow > 1) sheet.getRange(2, 1, lastRow - 1, 11).clearContent(); if (clientes.length > 0) { const filas = clientes.map((c, i) => [ c.n||i+1, c.nombre||'', c.pat||'', c.veh||'', c.tel||'', c.coch||'', c.horario||'', c.llave||'', c.estado||'Pendiente', c.monto||'', c.obs||'' ]); sheet.getRange(2, 1, filas.length, 11).setValues(filas); colorearSheet(sheet, clientes.length); } return responder({ ok: true, guardados: clientes.length }); } return responder({ ok: false, error: 'Acción no reconocida' }); } catch(err) { return responder({ ok: false, error: err.toString() }); } } function colorearSheet(sheet, total) { const estados = sheet.getRange(2, 9, total, 1).getValues(); for (let i = 0; i < total; i++) { let color = '#ffffff'; switch(estados[i][0]) { case 'Al dia': color = '#d9ead3'; break; case 'Pagó a Rojas': color = '#e9d0f7'; break; case 'Promo Rojas': color = '#fce5cd'; break; case 'Moroso': color = '#f4cccc'; break; case 'Baja junio': color = '#cfe2f3'; break; case 'Pendiente': color = '#f3f3f3'; break; } sheet.getRange(i + 2, 1, 1, 11).setBackground(color); } }
